BOM Studios — Design Guidelines (Final)

(Brand system you can hand to any designer or dev)

1. Colour Palette
Core

Deep Navy: #0F1638 (primary background)

Electric Blue: #394D9B (headlines, icons, accents)

Aqua Beam: #7DD1E7 (hover effects, micro-highlights)

Mint Lime: #A8D897 (success states, ticks)

Neutrals

Pure White: #FFFFFF

Soft Gray: #F0F2F6

Usage

Navy = base

Electric Blue = power colour

Mint = micro highlights

Aqua = motion/light effects

2. Typography
Primary Font: Inter

Clean, modern, SMB-friendly.

Secondary Mono (micro-use only):

IBM Plex Mono
For tiny labels, footers, and light “digital craft” accents.

Hierarchy

H1/H2: Inter Bold

H3: Inter Semibold

Body: Inter Regular

Microtext: IBM Plex Mono

3. Layout & Grid

8px base grid

Generous spacing

Section padding: 80px desktop, 48px mobile

Keep compositions clean, centre-aligned or left-aligned only

Use dotted pixel dividers between key sections

4. Visual Motifs
Warp-Speed Streaks

Low opacity (10–20%)

Used only in hero banner + rare section transitions

Pixel Corners

Applied to cards, buttons, or icons

Only 1–2px corner cuts for subtlety

Glows

Use sparingly

Aqua or blue glows for hover states only

5. Buttons
Primary Button

Electric Blue background

White text

Slight glow on hover

Pixel-corner or 1px offset shadow

Secondary Button

Transparent

Electric Blue border

Electric Blue text

6. Iconography

Simple geometric icons

Line weight 2px

Colour: Electric Blue

Optionally include tiny pixel dot highlights in Mint

7. Motion & Interaction

Hover glows

Subtle directional wipes

200–300ms transitions

Light ‘scanline’ hover on hero CTAs

8. Photo/Video Treatment

If using photography:

high contrast

neon rim-light

dark backgrounds

slight grain

If using motion backgrounds:

warp streaks

subtle noise

9. Logo Guidelines
Shape

BOM wordmark with bomb-O monogram

No gradients

Clean geometric bomb icon

Wide kerning

STUDIOS beneath in smaller uppercase, width-matched

Logo Colours

White on Navy

Electric Blue on Navy

Navy on Aqua (promo/merch)
